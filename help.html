Soyez informés du status des articles mentionnés dans votre document

A partir d'un document (< 2 Mo au format docx, odt ou pdf)

le programme recherchera dans le texte les références à des articles de code

selon le format d'annotation utilisé dans le texte:

  • ARTICLE - CODE: ex: "art. 1240 C. civ.", "article L. 110-1 du Code de commerce".
  • CODE - ARTICLE: ex: "C. civ. art. 1240 ", "Code de commerce article L. 110-1".
Consultez la liste des codes du droit français supportés

Il interrogera Légifrance à propos des articles détectés. En fonction de la plage temporelle choisie, vous saurez ainsi :

  • Si le texte est introuvable sur Légifrance (abrogé, faute de frappe, erreur de détection...)
  • Si le texte a été récemment modifié (période définie par vous) (N.B. : seule la modification la plus récente est mentionnée)
  • Si le texte va être modifié prochainement (période définie par vous) N.B. : seule la version à venir la plus proche est mentionnée)
  • Si le texte n'a pas subi de modification
